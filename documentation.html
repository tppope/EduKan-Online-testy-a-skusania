Technická dokumentácia

  • Aplikáciu je možné rozchodiť jednoducho nainštalovaním si aplikácie docker.
  • Následne v priečinku projektu spustiť v konzole príkaz docker-compose up --build -d.
  • Aplikácia bude bežať na portoch:
    • 8000 - Nginx server
    • 8080 - Phpmyadmin
    • 3306 - Mysql server
  • Adresa aplikácie bude localhost:8000
